## Features

- Fully responsive (desktop, tablet, mobile)
- Light & dark mode with a warm, editorial palette
- Smooth scrolling and subtle scroll-reveal animations
- Expandable experience timeline and project case-study modal
- Working contact form and one-click résumé download
- SEO meta tags + Open Graph for clean link previews
- **Single self-contained `index.html`** — all images, fonts, and the résumé PDF are embedded, so it works fully offline with zero dependencies

## Deploy to GitHub Pages

1. Create a repository (e.g. `portfolio`) and add `index.html` to the root.
2. Go to **Settings → Pages**.
3. Under **Source**, choose **Deploy from a branch** → `main` / `root` → **Save**.
4. Your site goes live in about a minute at `https://<username>.github.io/<repo-name>/`.

To use a custom domain, add it under **Settings → Pages → Custom domain**.

## Updating the site

`index.html` is a compiled, self-contained file — it is not meant to be hand-edited. To make changes, edit the source design and re-export a fresh `index.html`, then replace the file in the repo and push.

## Tech

Hand-crafted HTML, CSS, and vanilla JavaScript. No build step, no framework runtime, no external requests at load time.
